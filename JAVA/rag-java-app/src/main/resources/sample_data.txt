基本属性,氏名の入力フィールドは、姓と名の間に全角スペースを1つ入力する形式を推奨します。
基本属性,氏名のフリガナは、全角カタカナのみを受け付け、濁点・半角文字はエラーとなります。
基本属性,生年月日はカレンダー選択方式、または「YYYY/MM/DD」形式での直接入力が可能です。
基本属性,年齢は生年月日より自動計算され、読み取り専用フィールドにリアルタイムで表示されます。
基本属性,性別選択はラジオボタン形式で、未選択の状態では次へ進めないバリデーションがかかります。
住所情報,郵便番号を入力し検索ボタンを押すと、都道府県から町域までが自動補完されます。
住所情報,都道府県はプルダウン選択式とし、海外居住者の場合は「国外」を選択肢から選べます。
住所情報,住所の番地やビル名は、全角・半角のどちらで入力してもシステム側で正規化されます。
連絡先,電話番号はハイフン抜きの数字のみを入力し、10桁または11桁のチェックが行われます。
連絡先,メールアドレスは、RFC規格に準拠した形式チェックと、ドメインの有効性確認を行います。
アカウント,パスワードは英大文字、小文字、数字、記号を各1文字以上含む8文字以上が必要です。
アカウント,ユーザーIDはメールアドレスと共通化されており、登録後の変更はマイページから可能です。
勤務形態,雇用形態は正社員、契約社員、派遣社員、パートアルバイトの4種類から選択します。
勤務形態,出勤時刻と退勤時刻は、15分単位での切り捨て・切り上げ処理が自動適用されます。
勤務形態,休憩時間は、労働時間が6時間を超える場合は45分、8時間を超える場合は60分です。
金融情報,銀行口座の名義人は、通帳の表記通りに全角カタカナで入力する必要があります。
金融情報,振込先金融機関コードは4桁、支店コードは3桁の半角数字で入力してください。
金融情報,クレジットカード番号はセキュリティのため、下4桁を除きアスタリスクでマスクされます。
申請承認,申請フォームの提出後、上長による一次承認が完了すると自動的に人事部へ通知されます。
申請承認,差し戻しが発生した場合、申請者は修正後に「再申請」ボタンを押すことで処理を再開できます。
書類添付,添付ファイルの形式は、PDF、JPG、PNGのみ許可され、1ファイル最大5MBまでです。
書類添付,本人確認書類として、運転免許証またはマイナンバーカードの表面・裏面が必要です。
休暇管理,有給休暇の申請は、取得日の3日前までにシステム上で行うことがルールとなっています。
休暇管理,特別休暇（慶弔など）の申請には、証明となる書類の写しを添付する必要があります。
税金関連,源泉徴収票の再発行依頼は、システム内の申請ボタンより年間2回まで受け付けています。
税金関連,扶養家族の追加変更は、毎年11月の年末調整期間中に一括して受け付けます。
交通費,通勤交通費の計算は、自宅から事業所までの最短ルート（6ヶ月定期代）を基準とします。
交通費,近隣外出の交通費精算は、領収書が不要な公共交通機関に限り、実費精算が可能です。
汎用ルール,必須項目が未入力の場合、画面上部に赤いアラートメッセージが表示されます。
汎用ルール,下書き保存機能を利用すると、24時間以内であれば入力中のデータを再開できます。
